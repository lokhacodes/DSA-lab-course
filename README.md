# DSA LAB

This repository contains a collection of tasks for the Data Structures Lab course. Each task focuses on implementing various data structures and algorithms in C programming language.

## Tasks Overview

### Task-1: 1D Array Operations
- Date: 10-1-2023
- Topic: 1D Array
- Programs:
  - Storing and printing N elements in an array
  - Finding odd values in an array
  - Finding values at even indexes
  - Calculating sum of all elements
  - Calculating average of all elements
  - Copying elements from one array to another
  - Finding maximum value
  - Finding intersection of two arrays

### Task-2: Array Operations
- Date: 17-1-2023
- Topic: Array
- Programs:
  - Inserting an item at a specific index
  - Deleting an item from a specific index
  - Searching for a book ID (binary search)
  - Deleting an item by value
  - Bonus: Finding and deleting the 3rd odd value
  - Bonus: Inserting values before odd indexes

### Task-3: Sorting and Searching
- Date: 24-1-2023
- Topic: Array
- Programs:
  - Sorting students by height (ascending order)
  - Searching for book ID (binary search)
  - Finding seat number by ID after sorting
  - Finding index of mth largest value
  - Bonus: Inserting value at the index of mth largest

### Task-4: Stack Operations
- Programs:
  - Implementing stack with push and pop operations
  - Demonstrating stack functionality with various operations

### Task-5: Postfix Evaluation
- Programs:
  - Evaluating postfix expressions using stack

### Task-6: Queue Operations
- Programs:
  - Implementing circular queue with insert and delete operations
  - Demonstrating queue functionality

### Task-7: Additional Programs
- Programs: Mathematical Problems

### Task-8: Recursion
- Programs:
  - Calculating factorial using recursion

### Task-9: Additional Programs
- Programs: Additional issues 

### Task-10: Graph Algorithms
- Programs:
  - Adjacency matrix representation
  - Undirected graph implementation
  - Depth-First Search (DFS) traversal

## How to Run
Each task contains C source files (.c) that can be compiled and run using a C compiler like GCC.

Example compilation:
```
gcc filename.c -o output
./output
```
